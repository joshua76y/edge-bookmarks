🔄 同步 Edge 收藏夹

Edge 收藏夹文件路径：

%LOCALAPPDATA%\Microsoft\Edge\User Data\Default\Bookmarks

方法一：直接选择 Bookmarks 文件

  1. 按 Win+R，输入 %LOCALAPPDATA%\Microsoft\Edge\User Data\Default，回车
  2. 找到 Bookmarks 文件（无后缀名）
  3. 点击下方「选择文件」按钮，选中即可导入（仅导入收藏夹栏内容）

方法二：从 Edge 导出 HTML

  1. 在 Edge 中按 Ctrl+Shift+O 打开收藏夹
  2. 点击「导出收藏夹」，保存为 HTML 文件
  3. 在本工具中点击「导入」按钮，选择该 HTML 文件
