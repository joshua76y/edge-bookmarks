# Edge 收藏夹管理器

一个纯前端、零依赖的 Microsoft Edge 收藏夹管理工具。单个 HTML 文件，浏览器打开即用。

## 功能

- **导入收藏夹** — 支持 Edge 原生 `Bookmarks` JSON 文件和 HTML 导出格式，仅导入收藏夹栏内容
- **导入去重** — 按 URL 自动跳过已存在的书签，不产生重复
- **导出 HTML** — 导出为 Netscape Bookmark Format 标准 HTML 文件，可导入任何浏览器
- **最近使用** — 自动记录最近打开的 10 个书签，点击即可重新打开
- **搜索过滤** — 按书签名称或 URL 实时搜索
- **卡片式展示** — 书签以卡片网格形式展示，带 favicon 图标
- **可折叠文件夹** — 文件夹支持展开/折叠，展开状态自动保存
- **拖拽排序** — 拖动排序书签和文件夹，支持改变层级（拖入文件夹）
- **CRUD 操作** — 添加、编辑、重命名、删除书签和文件夹
- **分区管理** — 多分区展示（收藏夹栏、其他收藏夹等），支持一键删除空分区
- **数据持久化** — 所有数据存储在浏览器 localStorage 中，刷新不丢失

## 使用

直接用浏览器打开 `edge-bookmarks.html` 即可。

### 导入收藏夹

**方法一（推荐）：从 Edge 直接导入 JSON**
1. 按 `Win+R`，输入 `%LOCALAPPDATA%\Microsoft\Edge\User Data\Default`，回车
2. 找到 `Bookmarks` 文件（无后缀名）
3. 在工具中点击 **「同步 Edge」** → **「选择文件」**，选中该文件导入

**方法二：从 Edge 导出 HTML**
1. 在 Edge 中按 `Ctrl+Shift+O` 打开收藏夹
2. 点击「导出收藏夹」，保存为 HTML 文件
3. 在工具中点击 **「导入」**，选择该 HTML 文件

### 管理书签

| 操作 | 方式 |
|------|------|
| 打开书签 | 点击书签名称，自动记录到最近使用 |
| 编辑书签 | 鼠标悬停书签卡片，点击 ✎ 按钮 |
| 删除书签 | 鼠标悬停书签卡片，点击 ✕ 按钮 |
| 添加书签 | 点击顶部 **「+ 添加书签」** 按钮 |
| 新建文件夹 | 点击顶部 **「新建文件夹」** 按钮 |
| 重命名文件夹 | 鼠标悬停文件夹，点击 ✎ 按钮 |
| 删除文件夹 | 鼠标悬停文件夹，点击 ✕ 按钮 |
| 展开/折叠 | 点击文件夹名称 |
| 拖拽排序 | 鼠标悬停文件夹左侧 ☰ 手柄，拖拽到目标位置 |
| 删除分区 | 点击分区标题右侧 **「✕ 删除」** |
| 清理空分区 | 点击顶部 **「清理空分区」**（红色按钮） |

## 技术特性

- **纯前端** — 仅 HTML + CSS + JavaScript，零框架零依赖
- **单文件** — 所有代码在一个 HTML 文件中，JS 全部内联，无外部加载
- **零事件委托** — 所有动态按钮使用 `onclick` 内联属性，避免浏览器兼容性问题
- **无缓存问题** — 无外部 JS/CSS 文件，修改后刷新即可
- **localStorage 持久化** — 存储键名 `edge_bm_data`
- **兼容性** — 支持所有现代浏览器（Chrome、Edge、Firefox）

## 数据存储

所有书签数据存储在浏览器 `localStorage`，key 为 `edge_bm_data`。
清除浏览器数据或手动删除该 key 会丢失所有收藏夹数据，建议定期导出备份。
